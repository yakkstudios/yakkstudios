Metaplex MPL
Older SPL tokens launched before mid-2024. Uses Metaplex Token Metadata Program. Update via updateMetadataAccountV2.
Examples: most 2022–2023 tokens, manually created SPL tokens
Token-2022 / SPL-2022
New standard — pump.fun, most 2024+ tokens. Metadata stored in token extensions via TokenMetadata extension on the mint.
Examples: pump.fun tokens, Moonshot, most new launches
Not sure? Check Solscan → your token page → "Token Program" field. If it says TokenzQdBNbLqP5VEhdkAS6EPFLC1PHnBqCXEpPxuEb it's Token-2022. If it says TokenkegQfeZyiNwAJbNbGKPFXCWuBvf9Ss623VQ5DA it's MPL/SPL.
AUTHORITY CHECK: You must be the update authority of this token. For Token-2022 this means your wallet is the mint authority with metadata extension rights. For Metaplex MPL it's the update authority on the Metadata PDA. Both are verified on-chain when you sign — if not the authority the transaction fails and no charge is made.

  3. Metadata JSON built + pinned to IPFS

  5. Updates name, symbol, URI + additional fields in the mint extension data
  6. Broadcast to Solana mainnet — works with all Token-2022 aware explorers
